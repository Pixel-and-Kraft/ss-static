# SS-Static

Our starting point for platform-less static.

## Usage

1. `git clone https://github.com/Pixel-and-Kraft/ss-static.git`
2. `npm install`
3. `gulp watch`
4. open `http://localhost:3000`
5. have at it. enjoy. even have fun :)

**For the Curious:**  
The repo's "ss" is short for "Solid Start."
